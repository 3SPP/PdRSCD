# 传统处理方法

## 一、传统操作

传统操作方法为新增，用于一些网络最后仅返回单通道图像或在网络计算中需要，因此需要使用一些基本方法进行预处理或后处理，目前此处暂未分离，统一放于ppcd.traditions下，后期将分为预处理和后处理操作。

| 方法        | 简介                                           |
| ----------- | ---------------------------------------------- |
| CVA         | 计算两期图像的变化矢量                         |
| BaseCompute | 对两期图像进行做差/除运算                      |
| MRF         | 将原图和预测图输入马尔科夫随机场进行分割后处理 |

这里的传统操作主要用来辅助训练和预测，后期将分为预处理和后处理两部分操作。
